# Calculator Project

Yeh ek simple aur stylish calculator project hai jo basic web technologies ka use karke banaya gaya hai. Is project ka purpose browser me chalne wala ek working calculator banana hai jo clean design ke saath achha lage.

## Technologies Used

### HTML
`calculater.html` file project ka structure banati hai.

HTML ka use in cheezon ke liye hua hai:
- calculator ka main layout
- display section
- number aur operator buttons

### CSS
`Css/calculater.css` file project ki styling sambhalti hai.

CSS ka use in cheezon ke liye hua hai:
- modern background aur card design
- button colors aur spacing
- hover aur click effects
- responsive layout

### JavaScript
`Js/calculater.js` file calculator ki functionality handle karti hai.

JavaScript ka use in cheezon ke liye hua hai:
- button click events
- expression display karna
- calculation karna
- `Clear` aur `Del` button control
- result show karna

## Important Concepts Used

### DOM Manipulation
JavaScript me `getElementById()` ka use karke HTML elements ko select kiya gaya hai aur unka text update kiya gaya hai.

### Event Listeners
Har button par event listener lagaya gaya hai jisse click karne par corresponding action hota hai.

### Expression Evaluation
Calculation karne ke liye JavaScript ka `eval()` use hua hai.

Examples:
- `2+2`
- `8/4`
- `2**3`

## Features

- simple and clean calculator UI
- modern styled design
- number and operator buttons
- clear button
- delete button
- result display
- responsive design for smaller screens

## Files in This Project

- `calculater.html` - main structure
- `Css/calculater.css` - styling
- `Js/calculater.js` - functionality
- `project-details.md` - project explanation in simple language

## How to Run

1. Project folder open karo.
2. `calculater.html` file browser me open karo.
3. Calculator use karo.
