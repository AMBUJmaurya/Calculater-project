# Calculator Project Details

Yeh calculator project basic web technologies se banaya gaya hai. Is project ka main purpose ek simple aur achha dikhne wala calculator banana hai jo browser me chal sake.

## Kya-kya use hua hai

### 1. HTML
`calculater.html` file me calculator ka structure banaya gaya hai.

Isme:
- display area banaya gaya hai
- buttons banaye gaye hain
- calculator ka basic layout define kiya gaya hai

HTML ka kaam sirf page ka structure banana hota hai.

### 2. CSS
`Css/calculater.css` file me calculator ka design banaya gaya hai.

Isme:
- background color aur gradient use hua hai
- calculator box ko modern card style diya gaya hai
- buttons ko alag-alag colors diye gaye hain
- hover aur click effects diye gaye hain
- mobile screen ke liye responsive styling add ki gayi hai

CSS ka kaam page ko sundar aur attractive banana hota hai.

### 3. JavaScript
`Js/calculater.js` file me calculator ki functionality likhi gayi hai.

Isme:
- button click events handle kiye gaye hain
- numbers aur operators display me add kiye jaate hain
- `Clear` button pura expression hata deta hai
- `Del` button last character delete karta hai
- `=` button calculation karke result dikhata hai

JavaScript ka kaam calculator ko working banana hota hai.

## Project me aur kya use hua

### DOM Manipulation
JavaScript ke through HTML elements ko `getElementById()` se select kiya gaya hai aur unka text update kiya gaya hai.

### Event Listeners
Har button par click event lagaya gaya hai jisse user ke click par action hota hai.

### Eval Function
Expression calculate karne ke liye JavaScript ka `eval()` use hua hai.

Example:
- `2+2`
- `5*6`
- `2**3`

## Design Features

Is calculator me:
- centered layout hai
- dark modern background hai
- glassmorphism style card design hai
- soft rounded buttons hain
- alag colors for numbers, operators, action buttons aur equals button

## Summary

Is project ko banane me mainly yeh technologies use hui hain:

- HTML
- CSS
- JavaScript

Yeh ek front-end based calculator project hai jo browser me run karta hai aur kisi backend ki zarurat nahi hoti.
